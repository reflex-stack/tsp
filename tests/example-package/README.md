## Install

`npm i @reflex-stack/tsp-example`

## Usage

##### Main module
- `import { ... } from "@reflex-stack/tsp-example"`

##### Sub module
- `import { ... } from "@reflex-stack/tsp-example/submodule"`

## tsp commands

##### Build
- `npm run build`
##### Test
- `npm run test`
##### Publish
- `npm run publish`
